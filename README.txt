# LalaStreaming PRO - Listo para vender

## Archivos
- index.html = página pública para clientes
- admin.html = panel privado para editar
- styles.css = diseño
- data.js = datos iniciales
- app.js = lógica pública
- admin.js = lógica del panel

## Cómo probar
1. Abre `admin.html`
2. Contraseña inicial: 123456
3. Edita productos, precios, imágenes, WhatsApp y términos.
4. Guarda.
5. Abre `index.html` para ver la tienda como cliente.

## Link que debes enviar al cliente
Cuando lo subas a internet, envía solo el link de la página principal:
https://tudominio.com/index.html
o simplemente:
https://tudominio.com

No envíes `admin.html`.

## Importante
Esta versión está lista para usar en una PC o hosting estático, pero los cambios se guardan en el navegador donde editas.
Para que los cambios se vean desde cualquier dispositivo automáticamente, necesitas una base de datos real como Firebase.
